This repo contains various code related to generating audio to drive an
oscilloscope in X-Y mode.

Some interesting demos by other people:

wget http://mirror.kapsi.fi/koodaa.mine.nu/tvt/youscope-wave.wav

wget http://web.mit.edu/sdmorr/Public/oscillofun.flac
(for use with scope.py, open it in audacity, and export it as a wav file)
